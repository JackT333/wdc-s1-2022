Check out my borders

We've modified the content div to have a 1px darkgrey border with rounded corners of an 8px radius.

Other elements can have borders too, like this span with the class red-border.

You can change the color of horizontal rules by changing the border color;
all horizontal rules are now #9CFDFE4.

Alternately, this span has a red outline (red-outline class).
That prevents it from messing with the line height.
